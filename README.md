
# COMP-3018 Events API (Capstone)

Community **Events + RSVPs** backend API for **COMP-3018** using **Node.js + TypeScript + Express**.

## Links

- **GitHub repository (web):** https://github.com/Khushbhangu001/comp-3018-events-api  
- **GitHub repository (clone URL):** https://github.com/Khushbhangu001/comp-3018-events-api.git  
- **GitHub Project board:** https://github.com/users/Khushbhangu001/projects/1

## Git workflow

- **`main`**: stable, presentation-ready code  
- **`development`**: integrated ongoing work  
- **`feature/*`**: one branch per issue/feature (example: `feature/m1-venues-crud`)

## Tech stack 

- Node.js + TypeScript + Express  
- Firebase Authentication + Firestore (planned / in progress)  
- Joi validation  
- Swagger/OpenAPI (Milestone 2)  
- Jest + GitHub Actions CI (in progress)  
- dotenv + Helmet + CORS
